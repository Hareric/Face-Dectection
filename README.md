
## 使用方法：

用matlab打开该项目并导入路径后 运行gui.m

首先点击读入照片按钮并导入照片 然后点击人脸识别按钮


## 其他说明
该代码使用了matlab内置开源工具包

Computer Vision System Toolbox 

Image Processing Toolbox

需要使用2013版以后的matlab 程序运行，或者手动导入安装以上这2个包
